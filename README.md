# Classificador de Aprovação de Alunos

Projeto de Machine Learning usando Python e Scikit-Learn.

O objetivo é prever se um aluno será aprovado com base em:

- horas de estudo
- horas de sono
- faltas
- nota anterior
- participação

## Tecnologias

- Python
- Pandas
- Scikit-Learn

## Modelos testados

- Logistic Regression
- Decision Tree
- Random Forest
- KNN

## Avaliação

- Cross Validation
- Accuracy
